# terrain-generator
Proceduraly generates block-style terrain with OpenGL

## Libraries used
- GLFW 3.2.1
- Glad
- [nothings/stb](https://github.com/nothings/stb)
- GLM 0.9.7.6
- [Reputeless/PerlinNoise](https://github.com/Reputeless/PerlinNoise)

## Building
```
mkdir build
cd build
cmake ..
make
./terrain
```
